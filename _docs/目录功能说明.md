# 目录功能说明（当前版）

## `authoring/`

Blender 侧交互层，负责面板、操作、预览和导出。

## `components/`

组件与属性组织层，负责把 Blender 数据整理成 runtime 能用的结构。

## `runtime/`

运行时控制层，负责会话、桥接、输入整理、固定步调度和姿态输出。

## `_native/`

底层运行实现，负责更贴近 MC2 风格的数据流、碰撞处理和高频执行。

## `compile/`

构建与预编译辅助目录。

## `_docs/`

设计说明、历史记录、改造路线和踩坑总结。
